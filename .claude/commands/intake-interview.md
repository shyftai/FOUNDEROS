# FOUNDER:OS Intake Interview

Structured onboarding interview for founder workspaces. Ask one block at a time.

## Block 1 — Company Overview
- Company name?
- One-liner? (what you do, for whom)
- Stage? (idea, MVP, post-launch, growth, scale)
- Industry?
- Founded when?

Save to: `workspace.config.md`

## Block 2 — Team
- Co-founders? (names, roles)
- Team size?
- Key roles filled vs. open?
- Where is the team? (remote, office, hybrid)

Save to: `TEAM.md`

## Block 3 — Product
- What are you building?
- Current status? (idea, prototype, beta, GA)
- Tech stack?
- Key product milestones ahead?

Save to: `PRODUCT.md`

## Block 4 — Market
- TAM/SAM/SOM estimates?
- Competitive landscape? (top 3 competitors)
- Your differentiation?
- Market timing — why now?

Save to: `MARKET.md`

## Block 5 — Business Model
- Revenue model? (SaaS, marketplace, transactional, services)
- Pricing? (current or planned)
- Unit economics? (CAC, LTV, margins)

Save to: `BUSINESS.md`

## Block 6 — Funding
- Current runway? (months)
- Funding history? (rounds, amounts)
- Next raise planned? (when, how much)

Save to: `FUNDING.md`

## Block 7 — GTM
- Customer acquisition channels?
- Current traction? (users, revenue, growth rate)
- Sales process? (self-serve, sales-led, PLG)

Save to: `GTM.md`

## Block 8 — Connected OS Frameworks
- Which other OS systems are you using?
  - [ ] GTM:OS    [ ] CONTENT:OS   [ ] FINANCE:OS
  - [ ] HR:OS     [ ] INVESTOR:OS  [ ] COMMUNITY:OS
  - [ ] CS:OS     [ ] ADS:OS       [ ] LEGAL:OS
  - [ ] PRODUCT:OS

Save to: `CONNECTIONS.md`

## Block 9 — Decision Framework
- How do you prioritize? (gut, data, framework)
- Board involvement? (advisory, formal board)
- Decision-making speed? (fast and adjust, deliberate)

Save to: `DECISIONS.md`

## Block 10 — Goals
- Next 90-day goals? (top 3)
- Next major milestone?
- North star metric?

Save to: `ROADMAP.md`

---
After all blocks: display summary, suggest `/founder:dashboard` or `/founder:priorities`.
