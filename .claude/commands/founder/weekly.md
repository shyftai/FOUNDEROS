Weekly review — reflect on the past week, plan the next one. Pull data from all connected OS frameworks.

Display:
```
<< FOUNDER:OS // WEEKLY REVIEW >>
```

Follow these steps:

1. Pull this week's data from all connected OS frameworks
2. Ask the founder: "What were your top 3 wins this week?" — or suggest wins based on data
3. Ask: "What were the misses or things that didn't go as planned?"
4. Display the metrics comparison table (this week vs last week vs trend)
5. Review OKR progress — which key results moved? Which are stalling?
6. Review decisions made this week from DECISIONS.md — any outcomes to note?
7. Review delegation status from DELEGATION.md — anything overdue?
8. Review energy from ENERGY.md:
   - How many deep work hours were achieved vs target?
   - How many meeting hours?
   - Energy level this week?
   - What drained vs boosted energy?
9. Ask: "What are your top 3 priorities for next week?" — check they map to OKRs
10. Identify any items to escalate, park, or delegate
11. Extract learnings and log them to LEARNINGS.md
12. Update WEEKLY.md with the review content
13. Display the weekly summary:

```
  ┌─ WEEKLY SUMMARY ────────────────────────────────────────┐
  │                                                          │
  │  Week of: {date range}                                   │
  │                                                          │
  │  Wins:    {n}    Misses: {n}    Decisions: {n}          │
  │  Energy:  {score}/10     Deep work: {n}h / {target}h    │
  │                                                          │
  │  OKRs: {n} on track, {n} at risk, {n} behind           │
  │                                                          │
  │  Next week priorities:                                   │
  │  1. {priority}                                           │
  │  2. {priority}                                           │
  │  3. {priority}                                           │
  │                                                          │
  └──────────────────────────────────────────────────────────┘
```

14. End with:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Review saved to WEEKLY.md
  Learnings logged to LEARNINGS.md

  >> Next: /founder:today (Monday) · /founder:priorities

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

Best run on Friday afternoon or Sunday evening to close out the week and set up the next one.
