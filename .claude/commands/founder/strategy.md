Strategic planning — quarterly or annual strategy sessions.

Display:
```
<< FOUNDER:OS // STRATEGY >>
```

Follow these steps:

1. Ask: "What time horizon? Quarterly or annual?"
2. Read VISION.md, OKRS.md, PRIORITIES.md, ROADMAP.md, LEARNINGS.md
3. Pull current state from all connected OS frameworks:
   - GTM:OS → market position, pipeline health, campaign performance
   - FINANCE:OS → financial health, runway, revenue trajectory
   - HR:OS → team capacity, open roles, organizational health
   - INVESTOR:OS → fundraising position, investor sentiment
   - CONTENT:OS → brand position, content performance
   - COMMUNITY:OS → community health, engagement trends
4. Assess the current situation:
   - What's working? (double down)
   - What's not working? (fix or kill)
   - What's changed in the market? (adapt)
   - What have we learned? (from LEARNINGS.md)
5. For **quarterly strategy:**
   a. Review last quarter's OKR scores
   b. Identify the 3 biggest opportunities for next quarter
   c. Identify the 3 biggest risks for next quarter
   d. Draft 2-3 strategic themes for the quarter
   e. Map themes to OKRs (feed into /founder:okrs set)
   f. Identify resource needs (hiring, budget, tools)
   g. Update ROADMAP.md with quarterly milestones
6. For **annual strategy:**
   a. Review the full year — wins, misses, learnings
   b. Assess competitive landscape
   c. Identify strategic bets for next year (2-3 big bets)
   d. Define annual themes and milestone targets
   e. Map to quarterly breakdown
   f. Update ROADMAP.md with 12-month horizon
7. For each strategic bet, evaluate:
   - Thesis: why we believe this will work
   - Investment: what it costs (money, time, focus)
   - Expected return: what success looks like
   - Risk: what could go wrong
   - Review date: when to evaluate
8. Display the strategy summary:

```
  ┌─ STRATEGY — {period} ──────────────────────────────────┐
  │                                                          │
  │  Themes:                                                 │
  │  1. {theme}                                              │
  │  2. {theme}                                              │
  │  3. {theme}                                              │
  │                                                          │
  │  Strategic bets:                                         │
  │  1. {bet} — {expected return}                            │
  │  2. {bet} — {expected return}                            │
  │                                                          │
  │  Key risks:                                              │
  │  1. {risk} — mitigation: {plan}                          │
  │  2. {risk} — mitigation: {plan}                          │
  │                                                          │
  └──────────────────────────────────────────────────────────┘
```

9. Save strategy to ROADMAP.md and log decisions in DECISIONS.md
10. End with next steps: `/founder:okrs set` → `/founder:priorities`
