Display:
```
<< FOUNDER:OS // FEEDBACK >>
```

1. Ask: "What type of feedback? Bug / feature request / improvement / other"
2. Ask for details:
   - **Bug:** what happened, what you expected, steps to reproduce
   - **Feature:** what you need, why it matters, how you'd use it
   - **Improvement:** what exists, what could be better, your suggestion
3. Log the feedback with date, type, and details
4. Display:

