Retrospective on a quarter, initiative, or major event. Extract learnings and feed them forward.

Display:
```
<< FOUNDER:OS // DEBRIEF >>
```

Follow these steps:

1. Ask: "What are we debriefing? Quarter / initiative / launch / decision / other"
2. Ask: "What was the original goal or expected outcome?"
3. Pull relevant data from connected OS frameworks
4. Run the debrief framework:

   a. **What happened?** (facts only, no interpretation)
      - What were the outcomes vs expectations?
      - Pull actual metrics vs targets
   b. **What went well?** (amplify these)
      - What should we do more of?
      - What worked that we didn't expect?
   c. **What didn't go well?** (be honest)
      - What fell short and why?
      - Was it execution, strategy, timing, or external factors?
   d. **What did we learn?** (the real value)
      - What do we know now that we didn't know before?
      - What assumptions were proven right or wrong?
   e. **What will we do differently?** (make it actionable)
      - Specific changes to process, strategy, or approach
      - Who owns each change?

5. Categorize learnings for LEARNINGS.md:
   - Strategy learnings
   - Decision learnings
   - Delegation learnings
   - Domain-specific learnings (tag which OS)

6. Display the debrief:

```
  ┌─ DEBRIEF — {subject} ──────────────────────────────────┐
  │                                                          │
  │  Goal:     {original goal}                               │
  │  Outcome:  {actual outcome}                              │
  │  Grade:    {exceeded / met / missed}                     │
  │                                                          │
  │  Went well:                                              │
  │  1. {what worked}                                        │
  │  2. {what worked}                                        │
  │                                                          │
  │  Didn't go well:                                         │
  │  1. {what missed}                                        │
  │  2. {what missed}                                        │
  │                                                          │
  │  Key learnings:                                          │
  │  1. {insight → action}                                   │
  │  2. {insight → action}                                   │
  │  3. {insight → action}                                   │
  │                                                          │
  │  Changes going forward:                                  │
  │  1. {change — owner: {name}}                             │
  │  2. {change — owner: {name}}                             │
  │                                                          │
  └──────────────────────────────────────────────────────────┘
```

7. Log all learnings to LEARNINGS.md with proper categorization
8. Update any relevant DECISIONS.md entries with outcomes
9. End with:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Debrief complete. {n} learnings logged.

  >> Next: /founder:strategy · /founder:priorities · /founder:okrs

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
