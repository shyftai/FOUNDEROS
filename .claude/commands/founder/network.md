Manage advisor, mentor, and key relationships.

Display:
```
<< FOUNDER:OS // NETWORK >>
```

Follow these steps:

1. Read NETWORK.md
2. Determine the mode:
   - **Review:** Show all relationships and their status
   - **Add:** Add a new relationship
   - **Reach out:** Identify who to contact and draft outreach
   - **Ask:** Log a specific ask to the network
3. For **review:**
   a. Display advisors, mentors, peers, and key relationships
   b. Flag overdue touchpoints (last contact > expected cadence)
   c. Show active asks and their status
   d. Suggest who to reach out to this week based on cadence
4. For **add:**
   a. Ask: name, organization, relationship type (advisor/mentor/peer/key contact)
   b. Ask: what value they bring, what value you provide
   c. Ask: contact cadence (monthly/quarterly/annual)
   d. Add to NETWORK.md
5. For **reach out:**
   a. Identify contacts overdue for touchpoint
   b. For each, suggest a reason to reach out:
      - Share a win or milestone
      - Ask for specific advice related to current priorities
      - Offer value (intro, insight, resource)
      - Just check in (for close relationships)
   c. Draft a brief outreach message
6. For **ask:**
   a. Ask: "What do you need help with?"
   b. Scan NETWORK.md for who could help
   c. Suggest the best person to ask and why
   d. Draft the ask message
   e. Log the ask in NETWORK.md
7. Display network health:

```
  ┌─ NETWORK HEALTH ────────────────────────────────┐
  │                                                  │
  │  Advisors:      {n} ({n} overdue for contact)    │
  │  Mentors:       {n} ({n} overdue)                │
  │  Peer founders: {n} ({n} overdue)                │
  │  Key contacts:  {n} ({n} overdue)                │
  │                                                  │
  │  Active asks:   {n}                              │
  │  Open offers:   {n}                              │
  │                                                  │
  │  This week: reach out to {name}, {name}          │
  │                                                  │
  └──────────────────────────────────────────────────┘
```
