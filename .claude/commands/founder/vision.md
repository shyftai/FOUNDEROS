Define or refine the company vision, mission, values, and north star metric.

Display:
```
<< FOUNDER:OS // VISION >>
```

Follow these steps:

1. Read current VISION.md (if it exists)
2. Determine the mode:
   - **First time:** Full vision creation flow
   - **Refinement:** Update existing vision based on new context
3. For first-time creation, ask sequentially:
   a. "What does the world look like when your company succeeds?" — vision is a future state, not a task
   b. "How will you get there? What do you do, for whom, and why?" — this is the mission
   c. "What 3-5 values guide how you operate?" — values should be decision-making tools, not platitudes
   d. "What is the one metric that best captures value creation?" — the north star
   e. "What are you building that compounds over time?" — strategic moats
   f. "What does failure look like? What are you NOT building?" — anti-vision
4. For refinement:
   a. Display current vision, mission, values, north star
   b. Ask: "What has changed? What feels off?"
   c. Pull recent data from connected OS frameworks for context
   d. Propose updates with rationale
   e. Show diff between current and proposed
5. Validate the vision:
   - Is it inspiring enough to guide 3-5 years of decisions?
   - Is it specific enough to say no to things?
   - Does the north star metric actually capture progress toward the vision?
   - Are the values actionable in daily decisions?
6. Update VISION.md
7. Check if OKRs in OKRS.md still align with the updated vision — flag misalignment
8. Display:

```
  ┌─ VISION ────────────────────────────────────────┐
  │                                                  │
  │  Vision:     {statement}                         │
  │  Mission:    {statement}                         │
  │  North star: {metric} = {current}                │
  │  Values:     {v1}, {v2}, {v3}                    │
  │                                                  │
  └──────────────────────────────────────────────────┘
```

9. End with:

