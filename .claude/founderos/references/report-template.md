# Report Templates

Standard formats for FOUNDER:OS reports. Load before running `/founder:report`.

---

## Company performance report (internal)

```
# Company Performance Report — {Period}

## Executive summary
{2-3 sentence overview — state of the business in plain English}

## North star metric
{Metric}: {current value} (target: {target}, {trend}% change)

## Key metrics

| Category | Metric | Value | vs Last Period | vs Target | Status |
|----------|--------|-------|---------------|-----------|--------|
| Revenue | MRR | | | | |
| Revenue | Pipeline | | | | |
| Finance | Burn rate | | | | |
| Finance | Runway | | | | |
| Team | Size | | | | |
| Team | Open roles | | | | |
| GTM | Reply rate | | | | |
| GTM | Meetings | | | | |
| Community | Members | | | | |
| Community | Engagement | | | | |

## OKR progress
{Score each key result, highlight at-risk items}

## Highlights
1. {win}
2. {win}
3. {win}

## Challenges
1. {challenge — what we're doing about it}
2. {challenge — what we're doing about it}

## Key decisions this period
| Decision | Type | Outcome |
|----------|------|---------|

## Priorities for next period
1. {priority}
2. {priority}
3. {priority}

## Learnings
- {key insight}
```

## Weekly review format

```
# Weekly Review — Week of {date}

## Wins
1. {win — OKR link}
2. {win}
3. {win}

## Misses
1. {miss — root cause — action}
2. {miss}

## Metrics snapshot
| Domain | Key metric | This week | Last week | Trend |
|--------|-----------|-----------|-----------|-------|

## OKR check-in
| Objective | KR | Progress | Status |
|-----------|----|---------:|--------|

## Energy
- Deep work: {n}h / {target}h
- Meetings: {n}h
- Energy: {score}/10

## Next week priorities
1. {priority}
2. {priority}
3. {priority}
```

## Board update format

See `board-guide.md` for detailed guidance.

```
# Board Update — {Period}

## TL;DR
{3 sentences max}

## Highlights
- {milestone or win}

## Key Metrics
| Metric | Last Period | This Period | Target | Status |

## OKR Progress
{One line per KR}

## Challenges
- {challenge and what we're doing about it}

## Asks
- {specific, actionable request}

## Next Period Outlook
- {priority or milestone}
```
