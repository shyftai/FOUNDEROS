## Connected OS framework costs

FOUNDER:OS doesn't directly incur costs from connected frameworks — each OS tracks its own spend. However, FOUNDER:OS can aggregate cost data across all frameworks for a total view.

Check each framework's COSTS.md for detailed pricing:
- GTM:OS → Apollo, Instantly, enrichment tools
- CONTENT:OS → publishing tools, design tools
- FINANCE:OS → accounting tools, payment processing
- HR:OS → recruiting tools, HRIS
- INVESTOR:OS → CRM, data room tools
- COMMUNITY:OS → platform costs, engagement tools

---

## Cost tracking rules

1. Log every tool write in COSTS.md
2. Track aggregate spend across all frameworks monthly
3. Flag when total spend exceeds budget threshold (default: 80%)
4. Review tool ROI quarterly — cut tools that don't earn their keep
