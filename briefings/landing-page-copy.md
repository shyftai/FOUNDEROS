## Section 3: Solution Overview

**Heading:** One terminal. Eight operating systems.

Antigravity OS is a collection of open-source AI operating systems built on Claude Code. Each one handles a core business function -- from outbound sales to financial forecasting to fundraising. They run in your terminal. They connect to your existing tools. They enforce quality gates on every output. And they learn from your data.

No dashboards. No login screens. No onboarding calls with account executives. Just clone, onboard, execute.

FOUNDER:OS sits on top as the orchestrator -- pulling live data from all seven frameworks into a single founder cockpit. One command shows your pipeline, runway, open roles, investor pipeline, content calendar, and community health. All at once.

## Section 5: How It Works

**Heading:** Three steps. Five minutes. No sales call.

### Step 1: Clone the repo

```bash
git clone https://github.com/shyftai/GTMOS.git
cd GTMOS
claude
```

Pick any framework. Or pick all eight. Each one is a standalone repo.

### Step 2: Run onboard

```
/gtm:onboard my-company
```

Answer a few questions about your business. The system configures itself -- ICP, workflows, quality standards, connected tools.

### Step 3: Start executing

```
/gtm:new-campaign q2-enterprise
```

You're live. Every output goes through quality gates. Every action is logged. Every decision is traceable.

## Section 10: FAQ

**Heading:** Questions you're already asking.

### Is this really free?
Yes. Every framework is MIT licensed. Clone it, use it, modify it, ship it. There are no premium tiers, no usage limits, no "free trial." The code is yours.

### What do I need to run it?
Claude Code (Anthropic's CLI tool) and a terminal. That's it. Each framework is a standalone repo -- clone it, open it with Claude Code, and it boots automatically. No build step, no Docker, no infrastructure.

### Can my team use it?
Yes. Every framework supports team mode. Multiple people can work from the same workspace with role-based access, delegation tracking, and Slack notifications. Set it up during onboarding.

### Is my data safe?
Your data stays in your repo and your connected tools. Nothing is sent to Shyft AI servers. The frameworks run locally through Claude Code. You control what tools are connected, what APIs are called, and what data is accessed.

### What's the catch?
You need a Claude Code subscription from Anthropic, which is a paid product. The Antigravity OS frameworks themselves are free and open source. We make money from consulting and implementation services, not from the software.

### Can I customize it?
Everything is a markdown file. Every rule, every workflow, every quality gate, every default -- all overridable. Fork the repo. Change the rules. Add your own commands. The system is designed to be modified.
