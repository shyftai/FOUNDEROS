## Board update format

Every board update follows this structure:

### 1. Highlights (3-5 bullets)
- What went well this quarter
- Major milestones hit

### 2. Key metrics

| Metric | Last quarter | This quarter | Target | Status |
|--------|-------------|-------------|--------|--------|
| Revenue (MRR) | | | | |
| Pipeline | | | | |
| Burn rate | | | | |
| Runway | | | | |
| Team size | | | | |
| Key product metric | | | | |
| Customer metric | | | | |

### 3. Challenges (2-3 bullets)
- What's not working
- What you're doing about it

### 4. Asks (1-2 specific requests)
- Introductions needed
- Advice requested
- Decisions requiring board input

### 5. Next quarter outlook
- Top 3 priorities
- Key hires planned
- Expected milestones
