## Scoring guide

| Score | Meaning |
|-------|---------|
| 1.0 | Fully achieved |
| 0.7 | Strong progress, stretch target missed |
| 0.5 | Meaningful progress but short of target |
| 0.3 | Some progress, significant miss |
| 0.0 | No progress |
